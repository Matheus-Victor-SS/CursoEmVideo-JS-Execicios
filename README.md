# 📚 Curso em Vídeo - JavaScript (Exercícios)

Este repositório contém os exercícios que desenvolvi durante o curso de **JavaScript** do **Curso em Vídeo**, ministrado pelo professor Gustavo Guanabara.

## 🚀 Sobre o projeto

O objetivo deste repositório é registrar minha evolução no aprendizado de JavaScript, praticando conceitos fundamentais através de exercícios simples e projetos práticos.

## 📂 Estrutura do repositório

Os projetos estão organizados em pastas, cada uma representando um conjunto de exercícios ou um projeto específico:

* 📁 **Exercícios Simples**

  * Introdução ao JavaScript
  * Manipulação de dados
  * Condições e lógica
  * Eventos e DOM

* 📁 **Contador**

  * Projeto de contagem com interação do usuário

* 📁 **Horário Atual**

  * Exibe o horário atual com base no sistema

* 📁 **Tabuada**

  * Geração de tabuadas dinamicamente

* 📁 **Verificador de Idade**

  * Verifica idade e retorna informações com base no ano de nascimento

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📖 O que aprendi

Durante esses exercícios, desenvolvi habilidades como:

* Manipulação do DOM
* Uso de eventos em JavaScript
* Estruturas condicionais (`if/else`)
* Estruturas de repetição (`for`, `while`)
* Funções
* Interação com o usuário

## 🎯 Objetivo

Este repositório serve como prática e também como portfólio inicial dos meus estudos em programação.

## 📌 Observação

Os projetos são simples e focados no aprendizado. Conforme avanço nos estudos, pretendo melhorar e criar projetos mais completos.
