# 🕒 Projeto: Hora Atual Dinâmica

Este projeto foi desenvolvido com o objetivo de praticar **HTML, CSS e JavaScript**, criando uma página que mostra a hora atual em tempo real e muda dinamicamente o visual conforme o período do dia.

## 📌 Funcionalidades

* Exibe a **hora atual atualizada automaticamente**
* Mostra uma **mensagem personalizada** dependendo do horário:

  * Bom dia
  * Boa tarde
  * Boa noite
  * Boa madrugada
* Altera automaticamente:

  * **Cor de fundo (gradiente)**
  * **Imagem exibida**
* Atualização em tempo real a cada segundo

## 💡 Objetivo do projeto

Estou aprendendo desenvolvimento web e criando pequenos projetos para testar conceitos na prática.
Esse projeto foi feito para entender melhor:

* Manipulação do DOM com JavaScript
* Uso de condições (`if/else`)
* Trabalhar com datas e horários (`Date`)
* Alterações dinâmicas de estilo (CSS via JS)

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
